# houtai

## 项目介绍

图书馆后台管理界面，目前只有vue写的界面，无后台接口，后续会慢慢添加后台接口和页面

## 安装
```
yarn install
```

### 项目启动
```
yarn serve
```

### 项目构建
```
yarn build
```
